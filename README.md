# PhotoResizer

Bu Python projesi, bir klasördeki tüm resimleri belirli bir en-boy oranına göre yeniden boyutlandırır ve siyah arka plan ile doldurur.

## Kullanım

1. Python 3 yüklü olduğundan emin olun.
2. Gerekli kütüphane yükleyin:
```bash
pip install -r requirements.txt
```
3. `main.py` içindeki `input_folder` ve `output_folder` yollarını kendi klasörlerinize göre değiştirin.
4. En-boy oranını değiştirerek farklı oranlarda çıktı alabilirsiniz.
5. Programı çalıştırın:
```bash
python main.py
```
---
Tüm resimler `output_folder` klasörüne kaydedilecektir.
